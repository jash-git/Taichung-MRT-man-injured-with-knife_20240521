台中捷運男子持刀傷人釀三傷 乘客合力制伏 警查扣三把刀械

資料來源: https://tw.news.yahoo.com/%E5%8F%B0%E4%B8%AD%E6%8D%B7%E9%81%8B%E7%94%B7%E5%AD%90%E6%8C%81%E5%88%80%E5%82%B7%E4%BA%BA%E9%87%80%E4%B8%89%E5%82%B7-%E4%B9%98%E5%AE%A2%E5%90%88%E5%8A%9B%E5%88%B6%E4%BC%8F-%E8%AD%A6%E6%9F%A5%E6%89%A3%E4%B8%89%E6%8A%8A%E5%88%80%E6%A2%B0-034700829.html

台中捷運綠線今天上午約11時15分於列車上，發生男子持刀傷人案件，捷運公司立即將事發列車停在站內，並公告文心崇德站至文心櫻花站單線雙向運轉。凶嫌遭乘客合力制伏，共三人受傷，並查扣三把刀械，捷運市政府站已封鎖處理，犯案動機由警方調查釐清。

有網友今天在網路社群貼出影片，畫面可見中捷綠線開往北屯總站的列車上，行經市政府站時，一名戴帽子的男子持刀攻擊乘客，其他民眾見狀合力將男子制止。

台中市政府消防局表示，上午11時17分接獲中捷行控中心通報，捷運市政府站多名傷者需救護，派多輛救護車到場。

消防人員表示，三人送醫，包括一名17歲男性胸口、肩膀及手臂撕裂傷；一名27歲男性臉頰及下巴約15公分撕裂傷；而凶嫌本身右手指撕裂傷，都送往林新醫院。

台中市政府警察局第六分局說，初步調查，凶嫌在列車上持刀攻擊其他乘客，造成兩名乘客受傷。凶嫌被其他乘客制伏後，通報站務人員後送醫，目前初步查扣一把菜刀、兩把水果刀。

目擊乘客指出，凶嫌先把包包放在控制台上，突然從包包拿出刀子就砍人，離他最近的男子遭砍，菜刀被另一人奪下。

目擊者說，凶嫌又拿出其他刀子，接著車廂內其他乘客衝上去制伏凶嫌，大家不停敲凶嫌的頭，有人掰開凶嫌的手、奪下刀子，過程相當驚悚。

事件發生後，台中捷運在官網公告，文心崇德站到文心櫻花站單線雙向運轉；市政府站暫停營運。
